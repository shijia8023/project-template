# vueproj

> A project depend on Vue.js

## Build Setup

``` bash
# 安装依赖（初始化）
npm install --registry=https://registry.npm.taobao.org

# 热重载编译 localhost:8087
npm run dev

# 编译（无压缩）
npm run build

# 编译发布（压缩）
npm run publish

# 编译并查看分析报告（publish同理）
npm run build --report
```
#使用平台版本
version:1.3.3
